# VS Code Windows Shortcut Keys

Here are some common keyboard shortcuts for Visual Studio Code on Windows:

## General
- `Ctrl + Shift + P`: Open Command Palette
- `Ctrl + P`: Quick Open (Go to File)
- `Ctrl + Shift + N`: New Window
- `Ctrl + W`: Close Window
- `Ctrl + ,`: Open Settings
- `Ctrl + K Ctrl + S`: Open Keyboard Shortcuts
- `Ctrl + Shift + X`: Open Extensions
- `F11`: Toggle Full Screen
- `Ctrl + =`: Zoom In
- `Ctrl + -`: Zoom Out
- `Ctrl + 0`: Reset Zoom

## File Management
- `Ctrl + N`: New File
- `Ctrl + O`: Open File
- `Ctrl + Shift + O`: Open Folder
- `Ctrl + K Ctrl + O`: Open Folder in New Window
- `Ctrl + S`: Save
- `Ctrl + Shift + S`: Save As
- `Ctrl + K Ctrl + S`: Save All

## Editing
- `Ctrl + /`: Toggle Line Comment
- `Ctrl + Shift + /`: Toggle Block Comment
- `Alt + Shift + F`: Format Document
- `Ctrl + K Ctrl + F`: Format Selection
- `Ctrl + D`: Add Selection to Next Find Match
- `Ctrl + Shift + L`: Select All Occurrences
- `Ctrl + F`: Find
- `Ctrl + H`: Replace
- `Ctrl + Shift + F`: Find in Files
- `Ctrl + Shift + H`: Replace in Files

## Navigation
- `Ctrl + G`: Go to Line
- `Ctrl + Shift + O`: Go to Symbol
- `Ctrl + Tab`: Switch Between Open Tabs
- `Ctrl + Shift + Tab`: Switch Tabs in Reverse Order
- `Ctrl + Shift + T`: Reopen Closed Tab
- `Ctrl + K Ctrl + W`: Close All Tabs

## View
- `Ctrl + B`: Toggle Sidebar
- `Ctrl + Shift + E`: Show Explorer
- `Ctrl + Shift + F`: Show Search
- `Ctrl + Shift + G`: Show Source Control
- `Ctrl + Shift + D`: Show Debug
- `Ctrl + Shift + X`: Show Extensions
- `Ctrl + \``: Toggle Terminal
- `Ctrl + Shift + \``: Create New Terminal

## Debug
- `F5`: Start Debugging
- `Shift + F5`: Stop Debugging
- `F9`: Toggle Breakpoint
- `F10`: Step Over
- `F11`: Step Into
- `Shift + F11`: Step Out

## Tasks
- `Ctrl + Shift + P` then type "Run Task": Run Task
- `Ctrl + Shift + P` then type "Configure Task": Configure Task

For a complete list, refer to VS Code's official documentation or use `Ctrl + K Ctrl + S` to view all shortcuts.
